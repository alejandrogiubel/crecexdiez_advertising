# CreceXDiez Advertising
A CreceXDiez Advertising Platform Flutter plugin.

## Getting Started

### Add dependency

```yaml
dependencies:
  crecexdiez_advertising: ^0.2.0
```

### Just use it
```dart
import 'package:crecexdiez_advertising/crecexdiez_advertising.dart';
    Crecex10Advertising(
        id: 12,
        height: 100,
        width: 320,
        borderRadius: 10,
        adsIntervals: Duration(seconds: 10),
    ),
```
